# LightGate

A light weight API gateway

## Features

- [ ] Support JWT for authentication.
- [ ] Support forward signaling to another backend service.
- [ ] Support HTTP.
- [ ] Support HTTPS.
- [ ] Support WebSocket.
- [ ] Support WebSockets.
- [ ] Web UI.
- [ ] Support plugins through standard `net/http` middleware.
